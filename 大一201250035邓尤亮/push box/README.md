# 简介

- 一个简单的推箱子游戏，共3关

- 玩家需将所有箱子推入指定地点

- 游戏界面用字符来表示，图例如下

  | 字符 | 含义           |
  | ---- | -------------- |
  | █    | 障碍           |
  | ■    | 箱子           |
  | ◇    | 目标           |
  | ♂    | 人             |
  | ◆    | 已完成目标     |
  | ♀    | 在目标地上的人 |

- 使用方向键操控
- 按下大写 ‘R’ 键重新开始本关
